# Exceptions `notix.exceptions`
Exceptions are defined in **notix/exceptions.py** which is reference to all exceptions raised on error.

* `UrlPathException` - Exceptions is generated when you tried to create notix 404 urls using function [`get_url`](../../notix/get_url/).
